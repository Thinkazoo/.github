**Learning:** This repo is a special `.github` profile repo with no app code.

## 2026-01-12 - Semantic Hierarchy vs Visual Preference
**Learning:** READMEs often skip heading levels (H1 -> H3) to adjust font size, which breaks navigation for screen reader users.
**Action:** Always prioritize semantic structure (H1 -> H2 -> H3) over visual aesthetics in Markdown documents.
